當我安裝步驟執行到

```shell
npm install
```

會跳出很多錯誤訊息

<!--more-->

這才發現 Reveal.js 要用 python >= 2.5 and < 3.0.0
而我預設的 python 是 python3.5
所以只要像下面的指令把 python 指到 python2 就可以了

```shell
npm install --python=python2.7
```

`--python=` 後面是 python2 的執行擋，需要根據自己的設定做調整
