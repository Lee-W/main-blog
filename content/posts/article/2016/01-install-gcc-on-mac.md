最近用 brew 來安裝 gcc 5.3，一直卡在 `make --bootstrap`
放了幾個小時還是停在這

<!--more-->

之後才查到，要用 xcode 的命令來安裝
`xcode-select --install`

主要是因為 homebrew 下載的是還沒 compile 過的版本
而 compile gcc 需要非常長的時間
根據 Reference 中的文章，至少要超過 45 分鐘 ( 雖然我放了一天還是沒好 )
