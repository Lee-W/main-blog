前端工程的過去、現在與未來

slide

整體算是給完全不懂前端的我，關於前端的知識 overview

Present

  • Webpack 未來可能取代 Gulp 與 grunt
  • Babel: 目前最主流的 Javascript Transpiler
    • e.g. 轉換 ES6 → ES5
  • 雖然 ES6 支援模組標準，但瀏覽器尚未有定論，目前前端依然無法支援模組
    • 可以用 webpack 打包來模擬
