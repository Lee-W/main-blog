驅動BCM43228網卡在Linux的無線功能

Posted on 2015/01/28 - 三 in Linux-Unix

最近本來下定決心要來玩arch linux
結果竟然被筆電的網卡搞死了
 抓了官網的驅動竟然不能build…

沒想到改成灌Linux mint 17，依然有問題＝＝
最後找到一個解決的辦法

先確定你的網卡是不是BC43228

lspci | grep Network

如果是的話就能開始進行安裝了

sudo apt-get install linux-headers-generic
sudo apt-get install --reinstall bcmwl-kernel-source

如果成功的話，Wifi應該馬上就能用了

Reference

Ubuntu14.04 Acer Aspire V5 571 Broadcom BCM43228 WiFi - BeyondLogic
